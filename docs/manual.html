1紙娃娃換裝系統

角色是「紙娃娃」:一個底身(body)+ 可疊換的外觀層(髮色、上衣、褲子、帽子)。 每個外觀層都有走路與待機兩套動畫,換裝只是把對應圖層蓋上去,四個方向、每一幀都自動對齊。

怎麼換

  1. 打開遊戲頁,角色出現在畫面中央,右上角就是換裝面板。
  2. 面板分四組:髮色(原色 / 金 / 粉 / 銀)、上衣(原色 / 紅 / 藍 / 綠)、 褲子(原色 / 棕 / 綠)、帽子(無 / 棒球帽)。
  3. 點任一顆按鈕即時套用,選中的按鈕會亮成橘色;點「原色 / 無」還原該層。
換裝狀態會跟著你走。 走進別的房間、切換場景後,身上的裝扮不會被重置—— 因為換裝記在角色本身,不是記在房間裡。

想加新的顏色 / 新的部位?

底身之上的每一層都是獨立 sprite sheet,加新顏色是程式換色,不是重畫:

  • 髮色 / 衣色:用 tools/make-hair-overlay.py 從底身抽出該部位像素、換色生成新的 overlay。 (生圖模型畫不準對齊,所以換色一律走程式,不重畫。)
  • 生好的 sheet 命名成 char-<部位>-<顏色>-walk_sheet.png 與 ...-idle_sheet.png, 兩套都齊、寫進 assets/manifest.json,面板就會自動多出那顆按鈕。
注意:面板只列出「走路+待機兩套 sheet 都存在」的變體。只生了一半,那顆按鈕不會出現。

2布置一個房間

房間裡的家具都能用滑鼠直接拖。編輯模式下拖好位置,一鍵匯出這個房間的設定檔(JSON), 貼回專案就永久生效。全程不用寫程式。

操作流程

  1. 右上面板最下方「場景」區,點 ▶ 進入編輯模式。
  2. 按鈕變成綠色的 ■ 編輯中(點家具拖曳),下方多出「複製場景 JSON」。
  3. 用滑鼠點一件家具,它會亮成綠色(代表選中);按住拖動就能改位置。 放開後遮擋順序(誰在前誰在後)與碰撞範圍會自動跟著更新。
  4. 擺滿意後點 複製場景 JSON,整個房間的設定會複製到剪貼簿 (若瀏覽器擋剪貼簿,會改成下載一個 office.json)。

讓改動永久生效

匯出的 JSON 就是這個房間的完整定義。把它貼回對應檔案即可:

assets/scenes/<房間名>.json   # 例:office.json、cabin.json

下次重新載入,房間就是你剛剛擺好的樣子。JSON 內每件家具長這樣:

{ "asset": "office-chair", "x": 640, "y": 250, "flip": false }
  • asset:家具素材名(見附錄清單)。x / y:在房間內的座標(左上為原點)。
  • flip: true:水平翻面(椅子朝左朝右)。scale:單獨縮放這一件。
  • z:少數情況手動指定遮擋層(例如放在檯面上的小物要壓在檯子前面)。
加新家具進房間:不進編輯模式也行——直接在 JSON 的 objects 陣列裡新增一筆 { "asset": "...", "x": ..., "y": ... },存檔重整即可。想微調就再進編輯模式拖。

3做一張有多間房子、可進出的小地圖

整套地圖的骨架是「一個戶外中庭 + 幾間可進出的房間」。中庭(outdoor)上擺著幾棟房子外觀, 每棟房子門口設一個出入口(exit):角色踩上去就切進對應的室內房間;室內門口也設一個 exit 切回中庭。 這就是 gather 那種「地圖上有好幾間房、走進去就換場景」的做法。

目前的地圖結構

場景角色出入口通往
outdoor戶外中庭(樞紐)office / cabin / storage 三間房
office辦公室回 outdoor
cabin小木屋回 outdoor
storage倉庫回 outdoor

加第 4 間房(三步驟)

  1. 建室內場景檔 assets/scenes/lounge.json:複製一份現成的房間 JSON,改地板、牆、家具、 房間大小 size 與角色進場點 spawn。
  2. 在中庭放房子外觀 + 開一個出入口。 編輯 outdoor.json:
    • 在 objects 加一棟 house-exterior 當房子外觀。
    • 在 exits 加一筆,把房子門口的觸發區指向新房間:
    {
  "zone":  { "x": 600, "y": 525, "w": 90, "h": 30 },  // 門口觸發區(中心+寬高)
  "to":    "lounge",                              // 要進入的場景名
  "spawn": { "x": 450, "y": 500 }                 // 進去後角色出現在哪
}
  3. 在新房間開回程出入口。 在 lounge.json 的 exits 加一筆, to 設回 "outdoor",spawn 設成中庭那棟房子門口正下方, 這樣進出成對、走得回來。
出入口就是一個看不見的觸發框。 角色的碰撞框一旦碰到 zone,就立刻切到 to 場景、 落在 spawn。想「開門進屋」的感覺,就把 zone 貼在房子門口那一格。
務必成對設定。 只在中庭設了「進」、忘了在房間設「出」,角色會進得去出不來。 進與出的 spawn 要各自落在對方出入口旁邊一點(不要正好疊在觸發框上),否則會一進去又被彈回來、來回鬼打牆。
